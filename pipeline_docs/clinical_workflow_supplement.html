Step 1: Apply Clinical Rules

Module: APPLY_CLINICAL_RULES

Purpose: Match detected resistance markers against comprehensive clinical rules database

Input Data

  • Chromosomal mutations: TSV with detected resistance mutations
  • HGT genes: TSV with presence/absence of acquired resistance genes
  • Alleles: TSV with penA, porB, mtrR allele types
  • NG-STAR typing: TSV with 7-locus sequence type
  • Clinical rules JSON: Comprehensive rules database (clinical_rules.json)

Clinical Rules Database Structure

üî¥ CRITICAL Mutations (5 markers)

Highest-priority markers requiring immediate action:

  • penA A501V: >100-fold ‚Üë cephalosporin resistance
  • penA-60.001: FC428 clone marker (internationally spreading)
  • 23S C2611T: >1000-fold ‚Üë macrolide resistance
  • 23S A2059G: >1000-fold ‚Üë macrolide resistance
  • 23S A2058G: >1000-fold ‚Üë macrolide resistance

Actions: Immediate reporting, mandatory test of cure, treatment modification

üü† HIGH Priority Mutations (9 markers)

Significant resistance markers requiring monitoring:

  • penA G542S/G545S/A549S: 8-16-fold ‚Üë cephalosporin resistance
  • penA A510V/I515V/H541N: FC428 signature markers
  • gyrA S91F/D95G: >100-fold ‚Üë fluoroquinolone resistance
  • parC S87R: Synergistic with gyrA mutations

Actions: Standard reporting, test of cure recommended

üî¥ HGT Genes (6 genes)

Acquired resistance genes (plasmid-borne):

  • blaTEM: Œ≤-lactamase (penicillin resistance)
  • ermB/ermC/ermF: All macrolides ineffective
  • tetM: Tetracycline resistance
  • mefA: Reduced azithromycin susceptibility

Actions: Urgent reporting for erm genes, major public health concern

Synergistic Combinations

Output

  • Annotated data TSV: All resistance markers with clinical annotations
  • Alerts file: Priority alerts for critical findings

Step 2: Priority Classification

Module: PRIORITY_CLASSIFICATION

Purpose: Assign clinical priority levels based on resistance profiles

Priority Matrix (priority_matrix.tsv)

Defines priority assignment rules based on:

  • Number and type of critical mutations
  • Presence of HGT genes
  • Synergistic combinations
  • FC428 clone markers
  • XDR phenotype indicators

Priority Levels

Level Criteria Clinical Action Reporting
URGENT XDR phenotype (ESC + macrolide resistance) Immediate ID consult, inpatient therapy Immediate WHO notification
CRITICAL penA A501V or 23S mutations or erm genes Modified treatment, mandatory test of cure Immediate public health reporting
HIGH Multiple high-priority mutations or FC428 markers Enhanced surveillance, test of cure recommended Standard reporting
MEDIUM Single high-priority mutation Standard treatment, monitoring Routine surveillance
LOW Susceptible or minor markers only Standard dual therapy Routine reporting
Note: Priority classification considers both individual markers and their combinations. FC428 clone detection (penA-60.001 or A501V + A510V + I515V) automatically triggers CRITICAL priority.

Step 4: Clinical Report Generation

Module: GENERATE_CLINICAL_REPORT

Purpose: Create comprehensive HTML clinical reports for clinicians and public health officials

Report Components

  • Sample Information: ID, collection date, specimen type
  • Resistance Profile Summary: Visual dashboard of detected markers
  • Critical Alerts: Highlighted critical findings
  • Chromosomal Mutations: Table of detected mutations with MIC effects
  • HGT Genes: Acquired resistance genes detected
  • Allele Types: penA, porB, mtrR allele identification
  • NG-STAR Typing: 7-locus sequence type
  • Treatment Recommendations: Evidence-based therapy options
  • Priority Classification: Clinical urgency level
  • Public Health Actions: Required reporting and surveillance
  • Test of Cure Requirements: Follow-up testing guidance
  • References: Supporting literature and guidelines

Report Template

Uses HTML template (clinical_report_template.html) with:

  • Color-coded priority indicators
  • Interactive tables with sortable columns
  • Visual resistance profile charts
  • Printer-friendly format
  • CDC/WHO guideline references
Note: Reports are designed for both clinical use (treatment decisions) and public health use (surveillance and outbreak response).

Clinical Database Specifications

clinical_rules.json Structure

Top-Level Keys

  • critical_mutations: Highest priority markers (5 entries)
  • high_priority_mutations: Significant markers (9 entries)
  • synergistic_mutations: Dangerous combinations (4 entries)
  • hgt_genes: Acquired resistance genes (6 entries)
  • quality_control: QC markers (1 entry)

Entry Format

{
  "mutation_id": {
    "priority": "CRITICAL|HIGH|MEDIUM",
    "drug_affected": "Antibiotic class",
    "mic_effect": "Fold-change description",
    "treatment_modification": "Clinical action",
    "public_health_action": "Reporting requirement",
    "test_of_cure": "Mandatory|Recommended|Routine"
  }
}
